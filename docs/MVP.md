# MVP

## MVP Goal

Ship a first version that proves users want reference-driven music ideation before investing heavily in long-form generation.

## MVP Definition

Version 1 should focus on:

- project creation
- 2-3 reference inputs
- project settings
- AI analysis
- creative brief output
- project saving

## Core Inputs

Users can provide:

- reference link 1
- reference link 2
- reference link 3
- free-text theme
- optional lyrics direction
- BPM target
- duration target
- intensity or roughness target

## Core Outputs

The app returns:

- one-sentence track concept
- style blend summary
- BPM recommendation or confirmation
- mood description
- arrangement outline
- sound palette suggestion
- lyrics or hook seed
- generation-ready prompt

## MVP Screens

### 1. Home / New Project

Start a new project or open an existing one.

### 2. Project Workspace

Inputs for references, BPM, duration, theme, and mood.

### 3. Analysis Result

Shows the generated concept, arrangement, prompt, and direction.

### 4. Saved Projects

Browse and reopen previous work.

## Nice-To-Have But Not Required In MVP

- actual audio generation
- waveform previews
- accounts with social sharing
- public galleries
- payments
- advanced collaboration

## Optional Stretch For MVP

If feasible without derailing the project:

- short text-based iteration controls
- export project brief as markdown or text
- short prompt variations

## Success Criteria

The MVP is successful if users say:

- "This gave me a useful track idea fast."
- "The references were understood well."
- "The output felt musically relevant."
- "I would use this again while making music."
