# Roadmap

## Phase 0: Foundation

- lock product name and positioning
- define the first user workflow
- define legal boundaries for references
- choose stack and auth direction
- prepare repo docs and planning artifacts

## Phase 1: Smart Reference Workspace

Build the first valuable product layer:

- create project
- paste 2-3 references
- set BPM, duration, theme, and roughness
- run AI analysis
- return a creative brief
- save and revisit projects

This phase should prove the core insight:

mixed references can produce strong original direction.

## Phase 2: Guided Iteration

Add:

- prompt refinement
- alternate concept outputs
- stronger arrangement suggestions
- better lyrics and hook seeding
- style sliders such as more chaotic, more distorted, more melodic

## Phase 3: Preview Audio

Add:

- short audio sketch generation
- intro / drop / breakdown previews
- richer project output packaging

This is the first true audio-heavy step.

## Phase 4: Longer Generation

Add:

- 1 minute generation
- 3 minute generation
- 5 minute generation
- 7 minute generation

This phase is much more expensive and technically hard, so it should come only after the earlier workflow is strong.

## Phase 5: Iterative Editing

Add:

- continue track
- replace section
- alter energy curve
- adjust vocal mood
- regenerate around the same references

## Product Strategy Rule

Do not chase full "Suno competitor" status before the reference-driven core workflow is clearly valuable.
